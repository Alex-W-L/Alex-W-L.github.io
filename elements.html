Hi, I'm Alex

Interest in Data Analysis

My interest began relatively recently, in March of 2025. I spoke with a family member who manages data analysts on the topic and discovered that I already had some experience with a number of relevant tools. We talked about the field and I decided I wanted to learn more, so I did. I watched some video tutorials and followed along with guides to get some basics down, then I came up with a project idea. I wanted to do something local that was in an area that I felt was important. I happened upon PPB data from a data aggregation website and decided to run with my idea as a first project. I've worked for around 200 hours so far between learning more about Data Analytics libraries in Python, learning how to set up my own SQL server, learning how to use Tableau, and reviewing some Statistics from Khan Academy and Crash Course. I'm working on a Google Analytics certificate at the moment and trying to decide what my next project will explore.

Portland Crime Analysis

An analysis into crimes as reported by the Portland Police Bureau as part of the Portland City Open Data Initiative. Self-directed project involving data analysis with the Python libraries Pandas, GeoPandas, and Matplotlib as well as MySQL and with final displays using Tableau Public. In this Project I sought an answer to the questions: Is crime increasing in Portland as a whole? Is larceny increasing in areas where large businesses have closed?
